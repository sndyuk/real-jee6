Real-jee6

It is an explanation project of JEE6 and servlet 3.0.

View project onGitHub

Quick start

  1. Run hsqldb server.

    $ ./run-hsqldb-server.sh

  2. Open the other console.

  3. Install modules and Run glassfish server.

    $ ./run-server-local.sh

  4. Open web browser.

    http://localhost:8080/realjee/
