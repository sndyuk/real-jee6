Real-jee6

It is an explanation project of JEE6 and servlet 3.0.

View project onGitHub

Quick start

  1. Run hsqldb server.

    $ ./run-hsqldb-server.sh

  2. Open the other console.

  3. Install all modules

    $ mvn clean install

  4. Run server one of following:

    JSP + Servlet

    $ cd modules/real-jee-web
$ ./run-server-local.sh

    http://localhost:8080/realjee-web/

    JAX-RS

    $ cd modules/real-jee-rs
$ ./run-server-local.sh

    http://localhost:8080/realjee-rs/

    EAR

    1. Install modules and Run glassfish server.

      $ ./run-server-local.sh

    2. Wait until the console shows: This is not work! Please fix it.

      Hit ENTER to redeploy, X to exit

    http://localhost:8080/realjee-web/

Clean install all modules to Maven local repository

$ mvn clean install
